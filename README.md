# Rock Paper Scissors Game (C++)

An interactive command-line Rock Paper Scissors game developed in C++. This project simulates a classic game between the user and the computer, featuring multiple rounds, score tracking, and input validation.

## Features
- User vs Computer gameplay
- Multiple rounds with replay option
- Real-time score tracking system
- Randomized computer decisions
- Input validation for user choices
- Clear and user-friendly console interface

## Technologies Used
- C++
- Standard Library (iostream, cstdlib, ctime)

## Project Structure
- main.cpp → contains the game logic and functions

## How to Run
1. Compile the program:
   g++ main.cpp -o game
2. Run the executable:
   ./game

## Learning Objectives
This project was built to strengthen:
- Modular programming using functions
- Control flow (loops, conditionals, switch)
- Random number generation
- Passing variables by reference
- Writing clean and maintainable code
